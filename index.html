There was no focus on visual design, but on the implemented functionalities.

React / Chrome Extension

A currency converter Chrome extension, built using CSS HTML and React

The extension makes an API call to get all available currencies and displays them as a Select/dropdown menu. After selecting the base currency and the currency to convert to, an amount can be specified and the extension will make another API call and display the amount with 3 decimal points.
